### Section 1: Local Environment Setup

**Step 1: Check Java Version**
Ensure Java is installed on your system.
java -version

**Step 2: Install Maven**
Update your package list and install Maven.
Command: 
sudo apt update
sudo apt install maven -y

**Step 3: Create Maven Project**
Generate the project structure using the quickstart archetype.
Command:
mvn archetype:generate -DgroupId=com.example.app \
-DartifactId=demo-app \
-DarchetypeArtifactId=maven-archetype-quickstart \
-DinteractiveMode=false

**Step 4: Change Directory**
Command:
cd demo-app

**Step 5: Verify Files**
Command:
ls
Output: pom.xml, src

**Step 6: Project Structure**
demo-app/
├── pom.xml
├── src/
│   ├── main/java
│   └── test/java

**Step 7: Configure pom.xml**
Edit the file using: 
nano pom.xml
Content:

**Step 8: Write Application Code**
File Path: 
nano src/main/java/com/example/app/App.java
Code:
package com.example.app;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.apache.commons.lang3.StringUtils;

public class App {
    private static final Logger logger = LoggerFactory.getLogger(App.class);
    public static void main(String[] args) {
        String name = "CI/CD Pipeline";
        if (StringUtils.isNotBlank(name)) {
            String message = greet(name);
            logger.info(message);
            System.out.println(message);
        } else {
            logger.error("Name is empty!");
        }
    }
    public static String greet(String name) {
        return "Hello, " + name + "! Welcome to Maven CI/CD Demo.";
    }
}

**Step 9: Write Test Code**
File Path: 
nano src/test/java/com/example/app/AppTest.java
Code:
package com.example.app;
import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class AppTest {
    @Test
    public void testGreet() {
        String result = App.greet("Student");
        assertEquals("Hello, Student! Welcome to Maven CI/CD Demo.", result);
    }
}

**Step 10: Build Locally**
Command: 
mvn clean install

**Step 11: Run Application**
Command: 
mvn exec:java -Dexec.mainClass="com.example.app.App"

### Section 2: CI/CD Integration

**Step 12: Create Jenkinsfile**
Command: 
nano Jenkinsfile
Code:
pipeline {
    agent any
    tools {
        maven 'Maven3'
        jdk 'JDK21' 
    }
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/4kanksh/demo-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean compile'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
        stage('Package') {
            steps {
                sh 'mvn package'
            }
        }
    }
}

**Step 13: Push to GitHub**
Commands:
git init
git add .
git commit -m "Complete project for CI/CD"
git branch -M main
git remote add origin https://github.com/4kanksh/demo-app.git
git push -u origin main

**Step 14: Run Jenkins Pipeline**
1. Ensure Jenkins is running (java -jar jenkins.war).
2. Login to http://localhost:8080.
3. Manage Jenkins > Tools > Add JDK: Name "JDK21", Path "/usr/lib/jvm/java-21-openjdk-amd64".
4. Add Maven: Name "Maven3", Install automatically.
5. Create "New Item" > "Pipeline" named "demo-app-pipeline".
6. Configure SCM: Git, Repository URL, Branch "*/main".
7. Click "Build Now".
